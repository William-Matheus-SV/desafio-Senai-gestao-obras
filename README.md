# 🏗️ Sistema de Gestão de Obras

Sistema completo para gestão de obras da construção civil, desenvolvido em HTML, CSS, Bootstrap e JavaScript puro.

## 👥 Perfis de Acesso

| Perfil | Acessos |
|--------|---------|
| **👷 Mestre de Obras** | Checklist, Solicitar Material, Relatórios, Alocação de Funcionários |
| **📦 Almoxarife** | Solicitar Material, Armazém Central, Controle de Estoque |

## 🚀 Funcionalidades

- ✅ Checklist diário com 4 tópicos (Segurança, Materiais, Equipamentos, Ações)
- ✅ Solicitação e aprovação de materiais
- ✅ Armazém Central com controle de movimentações
- ✅ Estoque por unidade (Rua 1, Rua 2...)
- ✅ Alocação de funcionários (2 a 5 por obra)
- ✅ Relatórios enviados com histórico e exportação CSV
- ✅ Controle de permissões por perfil
- ✅ Persistência de dados via localStorage
- ✅ Design responsivo (funciona no celular)

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Vanilla)
- LocalStorage
- Git & GitHub

## 🎯 Objetivo do Projeto

Desenvolver um sistema integrado para gestão de obras da construção civil, permitindo:
- Controle de checklist diário
- Gestão de estoque e armazém
- Solicitação e aprovação de materiais
- Alocação de funcionários por obra
